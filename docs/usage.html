ユイシル α - 使い方

基本的な使い方

記録の追加

「贈った」または「受け取った」ボタンをクリックして、新しい記録を追加できます。

入力項目：
  • 相手の名前
  • 品名やカテゴリ（例：誕生日プレゼント、お土産など）
  • 日付
  • 詳細なメモ（贈り物の内容や思い出など）

関連する贈り物の設定

相手の名前を入力すると、その相手との過去の記録が表示されます。関連付けたい記録を選択することで、贈り物の関係性を記録できます。

注意： 贈った記録には受け取った記録を、受け取った記録には贈った記録のみを関連付けることができます。

記録の管理

各記録には以下の情報が表示されます：

  • 右上：相手との関係（例：「ユイシルさん へ贈った」）
  • 品名やカテゴリ
  • 日付
  • 詳細なメモ
  • 関連する贈り物（設定した場合）

便利な機能

相手別の絞り込み

画面上部の検索ボックスに相手の名前を入力すると、その相手との記録のみを表示できます。

記録の編集と削除

各記録の「編集」ボタンで内容を修正したり、「削除」ボタンで記録を削除したりできます。

注意： 記録を削除すると、関連付けも解除されます。

記録のシェア

新しい記録を保存すると、シェアモーダルが表示されます。以下の機能が利用できます：

  • 名前の表示/非表示の切り替え
  • メモの表示/非表示の切り替え
  • ランダムな感想文の再生成
  • X（旧Twitter）、Facebook、LINEでのシェア
  • テキストのコピー
ヒント： シェアテキストは、感謝の気持ちを伝えやすいように自動的にフォーマットされます。

データの保存

すべてのデータはブラウザのローカルストレージに保存されます。ブラウザのデータを削除すると記録も消えてしまうのでご注意ください。
