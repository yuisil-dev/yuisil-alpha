ユイシル α - 使い方

基本機能

ユイシルは、贈り物の記録を通して、人とのつながりや感謝の気持ちを可視化するサービスです。

以下の機能が利用できます：

  • 贈り物の送受記録
  • 相手別の一覧表示
  • 時系列での表示
  • 記録の編集・削除

使い方

1. 記録の追加

トップページの「贈った」または「もらった」ボタンをタップして、新しい記録を追加できます。

入力項目

  • 相手の名前（必須）
  • 日付（必須）
  • 内容（必須）

2. 記録の確認

追加した記録は、トップページの一覧に表示されます。

  • 贈った記録はオレンジ色のインジケーターで表示
  • もらった記録は水色のインジケーターで表示
  • 日付の新しい順に表示

3. 相手別の絞り込み

検索ボックスに相手の名前を入力すると、該当する記録のみを表示できます。

4. 記録の編集・削除

各記録の「編集」または「削除」ボタンから、記録の修正や削除ができます。

注意

削除した記録は復元できません。削除前に内容を確認してください。

データの保存について

ユイシルα版では、データはブラウザのローカルストレージに保存されます。

注意事項

  • ブラウザのデータを削除すると、記録も消去されます
  • 別のブラウザやデバイスでは記録を共有できません
  • プライベートブラウジングモードでは記録が保存されません
